# Pulse by Mrkters - Pitch Deck 🚀

This repository houses the interactive, web-based pitch deck for **Pulse by Mrkters**, prepared for the Wright Venture competition. 

Instead of a standard slide deck, we chose to build our presentation entirely in custom HTML, CSS, and JavaScript. This demonstrates our technical capabilities firsthand and perfectly matches the premium, high-stakes aesthetic of our brand.

### About Pulse
Local businesses are drowning in data but starving for speed. Pulse is an autonomous AI marketing analyst and receptionist built on a custom Python architecture. It answers inquiries in under 5 seconds, books appointments automatically, and provides dollar-for-dollar attribution for ad spend.
